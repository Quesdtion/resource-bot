# Resource Bot for Railway

## Переменные окружения

В Railway задай переменные окружения для сервиса бота:

- BOT_TOKEN — токен Telegram-бота
- DB_HOST — хост PostgreSQL (из Railway)
- DB_PORT — порт PostgreSQL (обычно 5432)
- DB_NAME — имя базы
- DB_USER — пользователь
- DB_PASS — пароль

## Что делает бот

- Выдаёт ресурсы (аккаунты) менеджерам
- Фиксирует состояние при получении (рабочий / в блоке / ошибка)
- Позволяет менеджеру отмечать срок жизни
- Ведёт историю операций в PostgreSQL
- Даёт отчёты:
  - /daily_report — общий
  - /manager_report — по конкретному менеджеру
  - /finance_report — финансовый (owner)

## Как запустить на Railway

1. Залей этот проект в GitHub (как репозиторий).
2. В Railway создай новый проект → "Deploy from GitHub".
3. В Railway добавь новый сервис PostgreSQL.
4. Скопируй параметры подключения БД в переменные окружения бота.
5. В переменных окружения также задай BOT_TOKEN.
6. Нажми Deploy.

Схема БД создаётся автоматически при первом запуске бота.
